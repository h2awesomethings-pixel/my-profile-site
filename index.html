Java/Spring 기반 백엔드 개발자로, 맡은 일은 반드시 해낸다는 것을 원칙으로 삼습니다. 외부 연동의 간헐적 지연을 전체 프로세스 검토 끝에 비동기 전환으로 해결하고, 느리던 주요 API를 밀리초 단위 응답까지 개선한 것처럼, 명확한 원인을 찾기 어려운 문제도 포기하지 않고 구조를 분석하며 실마리를 찾아왔습니다.

문제를 마주하면 누군가 해결해주길 기다리기보다 직접 원인을 분석하고 해결하는 것을 당연하게 여깁니다. 외부 솔루션 의존 기능을 자체 시스템으로 전환할 때는 탐지 프로세스 설계부터 API 개발, 유관 부서 협의까지 주도했습니다.

결제·인증 도메인에서 대규모 트래픽을 다루며 성능 개선, 신규 서비스 구축, 아키텍처 개선을 두루 경험했습니다. 서비스 구조와 데이터 흐름을 깊이 이해하고, 운영 중 발생하는 문제를 스스로 찾아 해결하는 개발자로서 팀에 기여하고 싶습니다.
